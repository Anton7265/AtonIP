Регистрация на GitHub и выкладывание сайта

Шаг 1: Регистрация на GitHub

Чтобы начать, вам нужно зарегистрироваться на GitHub. Нажмите кнопку "Sign up" и следуйте инструкциям для создания учетной записи.

Шаг 2: Создание нового репозитория

После регистрации, перейдите на главную страницу и нажмите на кнопку "New" для создания нового репозитория.

  1. Введите название репозитория (например, my-website).
  2. Выберите Public для открытого репозитория.
  3. Отметьте опцию "Initialize this repository with a README".
  4. Нажмите кнопку "Create repository".

Шаг 3: Добавление файлов

Теперь вы можете добавить файлы для вашего сайта:

  1. Нажмите на кнопку "Add file" и выберите "Upload files".
  2. Перетащите файлы вашего сайта в окно загрузки или выберите их с вашего компьютера.
  3. Нажмите "Commit changes" внизу страницы после загрузки файлов.

Шаг 4: Публикация сайта с помощью GitHub Pages

Чтобы опубликовать ваш сайт, выполните следующие шаги:

  1. Перейдите в настройки вашего репозитория, нажав на вкладку "Settings".
  2. Прокрутите вниз до секции "GitHub Pages".
  3. В разделе "Source" выберите ветку main и нажмите "Save".

Теперь ваш сайт доступен по адресу: https://<ваше-имя-пользователя>.github.io/<название-репозитория>/

Шаг 5: Изменения и обновления

Чтобы внести изменения на ваш сайт, просто добавьте новые файлы или измените существующие, затем сделайте commit и изменения будут отображены на вашем сайте через некоторое время.
